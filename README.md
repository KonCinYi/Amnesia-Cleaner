**Автоматическая утилита для системных администраторов.**  
Предназначена для обслуживания общественных компьютеров (компьютерные классы, офисы, интернет-кафе, коворкинги). 

Скрипт одной кнопкой очищает временные файлы всех пользователей и выполняет **полный сброс** браузеров до заводских настроек, удаляя следы предыдущих сессий.

---

## 🚀 Возможности

*   **⚡ Полная автоматизация:** Скрипт сам запрашивает права Администратора и обходит всех пользователей в системе.
*   **🛡️ Безопасная очистка Temp:**
    *   Удаляет **только файлы**, не трогая структуру папок (чтобы не ломать софт).
    *   Удаляет только мусор **старше 5 дней** (активные сессии и недавние документы остаются целы).
*   **🌐 Глобальный сброс браузеров:**
    *   Поддерживает **20+ браузеров** (Chromium, Firefox, Opera, Yandex и др.).
    *   Удаляет кэш, куки, историю, сохраненные пароли и сессии.
    *   Браузер запускается как при первом старте (Welcome Screen).
*   **📂 Чистота:** Работает через гибридный `.bat` файл (не требует установки, не детектится антивирусами как `.exe`).

## 📥 Установка и запуск

1.  Скачайте файл [`Amnesia.bat`](./Amnesia.bat) из этого репозитория.
2.  На целевом компьютере кликните по файлу **дважды** (Double-click).
3.  Разрешите запуск от имени Администратора в появившемся окне.
4.  Дождитесь сообщения:
    > `УСПЕХ! Мусор удален, браузеры сброшены, система цела.`

---

## ⚙️ Настройка

Вы можете изменить порог "старости" временных файлов, отредактировав одну строчку в файле `Amnesia.bat`:

Порог безопасности для Temp (файлы старше 5 дней)
$limitDate = (Get-Date).AddDays(-5)

- Измените -5 на -1, чтобы удалять всё, что старше суток.
- Измените на -30, чтобы проводить только ежемесячную чистку.
